# Skills & Technology Stack

## Core Framework
- **React (Vite):** Use functional components and Hooks (`useState`, `useEffect`, `useRef`).
- **Language:** TypeScript (Strict mode).

## Styling & UI
- **Tailwind CSS:** For layout and typography.
- **Shadcn/UI:** For aesthetically pleasing input sliders, cards, and buttons.
- **Lucide React:** For icons.
- **Framer Motion:** For smooth transitions and animations (if needed for UI elements).

## Graphics & Simulation
- **HTML5 Canvas API:** Preferred over SVG for high-performance rendering of rays and dragging logic.
- **Math:** Native JavaScript `Math` object. Ensure precise floating-point handling for physics calculations.

## Capabilities
- **Physics Engine:** Implement accurate Ray Optics physics (Snell's Law).
- **Interactive UI:** Implement mouse/touch listeners for "drag-to-change-angle" functionality.
- **Responsive Design:** Simulation must resize correctly on window resize.
